# KM Blog - Multiuser blog
Simple ecommerce app created using nestjs and reactjs.

## Features
- Login / Register
- Create / Upate / Delete product for admin
- review a product
- Browse product by category
- Responsiveness

## Setup
To run this project, install it locally using yarn:

```
$ cd ../backend
$ yarn install
$ yarn run start:dev
$ cd ../frontend
$ yarn install
$ yarn start
```
